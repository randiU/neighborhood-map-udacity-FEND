# Google Maps Neighborhood React Project

## Project Overview

This is a project from the Udacity Front-End Nanodegree program that uses React to build a website and utilizes API requests to get the information used on the site.

## App Description

The user can filter the venues listed on the Google Maps section of the page.

## Installation

Download the project repo and install npm by running `npm install` in the terminal. Then you can start the server by running `npm start` .

To run the app in production build:

`npm run build`

`serve -s build`

and then visit the localhost indicated. 

## Built With 

- HTML
- CSS
- Javascript
- React 
- Google Maps API
- Foursquare API
